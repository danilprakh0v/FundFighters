# FundFighters 🗡️💰

**FundFighters** — это не просто финансовый трекер. Это геймифицированная RPG-система учета личных финансов, где каждая накопленная копейка наносит урон "Монстру", олицетворяющему вашу финансовую цель! Удобный и чистый iOS-интерфейс, совмещенный с C# ASP.NET Core бэкендом.

---

## 🎯 Назначение проекта
Проект разработан в рамках курсовой работы (Дисциплина ФКН ВШЭ "НИС"). Основная идея — решить проблему потери мотивации в долгосрочном накоплении денег путем добавления игровых механик: очки опыта (XP), битвы с "Немезидами" (целями), красочные дэшборды с достижениями, и т.д.

## 🚀 Основные фичи продукта
- **Авторизация и безопасность**: Регистрация с подтверждением по Email (SMTP) и надежная защита паролей.
- **Геймификация Сбережений (Savings Goal & Battle System)**: Вместо бара "Вы накопили 30%" игрок видит босса (например, PlayStation 5), которому нужно "снести" HP, переводя средства в накопления. Как только HP монстра достигает 0 — цель достигнута!
- **Аналитика (Expense Chart)**: Интеллектуальная разбивка категорий расходов через кастомные Breakdown-бары.
- **Недавняя активность (Recent Activity)**: Трекинг доходов и расходов с фильтрацией по дате и пружинными микро-анимациями действий.
- **Pixel-perfect Design**: Написанный полностью кодом (без Storyboards) UI с использованием единого источника стилей — Design System (`DS`).

## 🛠 Технологический стек
**Мобильный клиент (iOS)**
- **Swift / UIKit**: Вся верстка выполнена сугубо кодом для обеспечения высшего контроля над рендерингом и констрейнтами.
- **Архитектура**: MVVM (Model-View-ViewModel) вкупе с неявным Coordinator. Разделение бизнесс-логики от UI.
- **UI Слои**: Продвинутые `CAGradientLayer`, `UIView.animate` с пружинной физикой, сложные переходы между экранами (Splash -> Entry -> Auth -> Dashboard -> Battle).

**Backend (REST API)**
- **C# / ASP.NET Core**: Надежный API слоем контроллеров и сервисов логики.
- **Entity Framework Core (PostgreSQL)**: ORM и миграции базы данных.
- **Clean Architecture / CQRS**: Логика разделена на Presentation, Application, Domain, Infrastructure.
- **SMTP Service**: Отправка Email-кодов.
- **JWT Authentication**: Безопасный токенизированный доступ (в разработке / заготовка).

## 📥 Запуск
1. Склонируйте репозиторий.
2. Для бэкенда: обновите `appsettings.json` (ConnectionString, SmtpSettings) и выполните миграции EF Core. Нажмите Run.
3. Для клиента: Откройте `.xcworkspace` (или просто папку в JetBrains Rider), выберите симулятор (15 Pro) и нажмите Run. Пакетные менеджеры (SPM/Cocoapods) на текущем этапе сведены к минимуму или не требуются (полагаемся на Foundation/UIKit).
